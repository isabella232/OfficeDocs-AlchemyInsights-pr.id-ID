# <a name="block-pop-up-windows-in-microsoft-edge"></a>Memblokir jendela pop-up di Microsoft Edge

1. Masuk ke **... (Pengaturan dan lainnya)**  >  **Pengaturan**  >  **Izin situs**.
2. Pilih **pop-up dan pengalihan**.
3. Aktifkan **blokir**.
