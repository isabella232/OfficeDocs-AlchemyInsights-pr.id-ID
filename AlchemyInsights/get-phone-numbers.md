# <a name="get-phone-numbers"></a>Mendapatkan nomor telepon

Setelah membeli [lisensi Telepon Microsoft Teams dan Paket Panggilan Microsoft Teams](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system#step-2-buy-and-assign-phone-system-and-calling-plan-licenses), Anda perlu mendapatkan nomor telepon untuk pengguna Anda. Ada 3 cara untuk mendapatkan nomor telepon.

1. [Mendapatkan nomor baru menggunakan pusat admin Teams](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system#get-new-user-phone-numbers-using-the-teams-admin-center).
2. [Mendapatkan nomor baru yang tidak tersedia di pusat admin Teams](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system#get-new-numbers-that-arent-available-in-the-teams-admin-center).
3. [Memindahkan atau mentransfer nomor telepon dari penyedia layanan atau operator telepon](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system#port-or-transfer-phone-numbers-from-your-service-provider-or-phone-carrier).

Untuk informasi selengkapnya, lihat [Mendapatkan nomor telepon untuk pengguna Anda](https://docs.microsoft.com/MicrosoftTeams/setting-up-your-phone-system#port-or-transfer-phone-numbers-from-your-service-provider-or-phone-carrier).
