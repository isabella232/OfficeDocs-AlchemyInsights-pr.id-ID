# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload ke OneDrive atau SharePoint

Anda dapat mengunggah file ke pustaka dokumen SharePoint Online cukup dengan menyeretnya dari komputer dan meletakkannya ke pustaka dokumen. Dengan OneDrive, Anda bisa menyinkronkan file antara komputer Anda dan awan, sehingga Anda bisa masuk ke file Anda dari mana saja. Anda juga dapat bekerja dengan file yang disinkronkan secara langsung di File Explorer dan mengaksesnya bahkan ketika offline. Untuk informasi selengkapnya tentang mengunggah dan menyinkronkan, silakan kunjungi tautan di bawah ini.
