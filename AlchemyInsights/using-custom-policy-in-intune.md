# <a name="using-custom-policy-in-intune"></a>Menggunakan kebijakan kustom di Intune

Intune memiliki banyak pengaturan bawaan untuk perangkat Android, iOS, macOS Windows. Anda juga dapat membuat profil kustom lalu menyebarkan pengaturan kustom ini ke perangkat menggunakan Intune. Profil kustom dirancang untuk menambahkan pengaturan dan fitur perangkat yang tidak dikembangkan di Intune.

[Pengaturan kustom untuk perangkat Android di Intune](https://docs.microsoft.com/intune/custom-settings-android)

[Pengaturan kustom untuk perangkat Android Enterprise di Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[Pengaturan kustom untuk perangkat iOS di Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[Pengaturan kustom untuk perangkat macOS di Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[Pengaturan kustom untuk Windows 10 di Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[Pengaturan kustom untuk Windows Holographic for Business di Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
