# <a name="microsoft-365-apps-crash-stop-responding-or-dont-launch"></a>Microsoft 365 aplikasi mengalami crash, berhenti merespons, atau tidak diluncurkan

Untuk memperbaiki Microsoft 365 aplikasi mengalami hang atau crash, cobalah opsi berikut ini:

- [Selidiki kemungkinan adanya masalah dengan add-in](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d#bkmk_addins) yang diinstal untuk Office
- Memeriksa apakah [perangkat lunak antivirus berkonflik](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d?ocmsassetID#bkmk_conflict) dengan Microsoft 365 antivirus
- Memverifikasi atau [menginstal pembaruan terkini untuk Office](https://support.office.com/article/update-office-and-your-computer-with-microsoft-update-2ab296f3-7f03-43a2-8e50-46de917611c5)
- [Memperbaiki Office](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b) menggunakan Panel Kontrol
- [Hapus instalan](https://support.office.com/article/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8) dan instal Office

Untuk informasi selengkapnya, lihat:
- [Excel merespons, berhenti, beku atau berhenti bekerja](https://support.office.com/article/excel-not-responding-hangs-freezes-or-stops-working-37e7d3c9-9e84-40bf-a805-4ca6853a1ff4)
- [PowerPoint tidak merespons, berhenti atau beku](https://support.office.com/article/powerpoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d)
- [Saya mendapatkan kesalahan "berhenti berfungsi" ketika memulai Office aplikasi di PC](https://support.office.com/article/i-get-a-stopped-working-error-when-i-start-office-applications-on-my-pc-52bd7985-4e99-4a35-84c8-2d9b8301a2fa)
- [Membantu melindungi file Anda jika terjadi crash](https://support.office.com/article/help-protect-your-files-in-case-of-a-crash-551c29b1-6a4b-4415-a3ff-a80415b92f99)
